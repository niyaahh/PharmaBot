# 💊 PharmaBot – Automated Medicine Dispensing System

PharmaBot is an IoT-based automated medicine dispensing system designed to improve the efficiency and accuracy of medicine distribution. The system uses QR code-based prescription verification, an ESP32 microcontroller, servo motors, and a MySQL database to provide secure and contactless medicine dispensing.

---

## 📌 Features

- QR code-based prescription verification
- Automated medicine dispensing using servo motors
- Secure prescription management
- MySQL database integration
- User-friendly doctor interface
- Real-time medicine dispensing
- Contactless medicine collection

## ⚙️ How It Works

1. Doctor logs into the system.
2. A digital prescription is created.
3. A QR code is generated for the patient.
4. The patient scans the QR code at PharmaBot.
5. The ESP32 verifies the prescription.
6. Corresponding servo motors dispense the prescribed medicines.
7. LCD displays the dispensing status.
8. After completion, the system displays **"THANK YOU"**.
